- What's the largest decimal number you can store in 5 digits of binary?
  - Largest decimal number that can be represented with 5 binary digits is 31. 
  - Formula -> `$2^n - 1$`
  - The binary maximum value is represented as 11111 -> 16 + 8 + 4 + 2 + 1
