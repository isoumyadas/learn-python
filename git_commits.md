## Common commit types

feat => New feature
fix => Bug fix
docs => Documentation only
style => Formatting (spaces, semicolons, linting)
refactor => Code change without bug fix or feature
test => Adding or updating tests
chore =>Maintenance (deps, configs, CI)

## Examples (industry-ready):

feat: add user profile page

fix(auth): resolve token expiration issue

docs: update API usage examples

refactor: simplify order processing logic

test: add unit tests for payment service

chore: upgrade lodash to latest version
